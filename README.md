# Minecraft-Tools
A minecraft bot where you can keep track of all ores and seeds for your world and see your progress and also share your inventory 



Install: 
Configure the index.js at the top and deploy-commands.js

In your terminal
npm i discord js
npm i ms

Running the bot
node deploy-commands.js
node index.js

generate-commands.js is used per say you broke a command file and dont know how to fix it you can do node generate-commands and itll update all commands
